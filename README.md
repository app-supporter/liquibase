# 🐳 CICD.

PR이 생성되면 코드 **`정적 분석`** 을 합니다. 코드 리뷰를 거친 후, dev 브랜치로 병합이 되면 dev 서버에 배포가 되며, QA과정을 거친 후, prod 서버로 배포가 이루어 집니다.
